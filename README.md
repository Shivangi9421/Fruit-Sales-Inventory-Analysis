# Fruit-Sales-Inventory-Analysis
This repository contains a professionally designed Power BI (.pbix) report focused on analyzing sales performance and inventory health.The report integrates key operational metrics to help businesses gain visibility into product demand, stock movement, and revenue contribution.

#Key Features

1.Comprehensive multi-page Power BI report

2.Combined sales + inventory perspective

3.Interactive filtering and drill-down functionality

4.KPIs for revenue, stock levels, demand variance, and product ranking

5.Clear visual hierarchy and modern dashboard layout

6.Optimized DAX calculations for accuracy and performance

#Detailed Insights Included
1. Sales Metrics
Total and monthly revenue
Regional sales performance
Top-selling products
Customer demand trends

2. Inventory Performance
Current stock levels
Stock-out & overstock alerts
Inventory movement analysis
Warehouse performance indicators

3. Product Analysis
Category and subcategory insights
Profit contribution
Sales vs. stock correlation
SKU-level metrics

4. Combined Sales & Inventory View
Demand vs. availability gaps
Replenishment recommendations
Trend forecasts
Product lifecycle evaluation
